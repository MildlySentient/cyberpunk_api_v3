Privacy Policy for Cyberpunk 2020 GPT Tools

Last updated: 2024-05-20

1. Data Collection

This application (Cyberpunk 2020 GPT API and Custom GPT) does not intentionally collect, store, or share any personal information about users. All interactions with the API and GPT are transient and are not saved by the backend or the developers.

2. Data Processing

  • Queries sent to the API or GPT are processed in memory to generate a response and are not logged or stored.
  • Uploaded files (e.g., game data, scenario tables) are used only to generate immediate responses and are not retained.

3. Third Parties

  • The backend is hosted on Render.com or Railway and subject to their privacy and security policies.
  • No analytics, tracking, or advertising services are used.

4. Security

Reasonable efforts are made to secure the server and codebase, but no system is perfectly secure. Do not submit sensitive or personal information through this tool.

5. Children's Privacy

This tool is not intended for children under 13. No information is knowingly collected from minors.

6. Changes to This Policy

This policy may be updated at any time. Changes will be published to the public GitHub repository or project domain.

7. Contact

For questions or concerns, contact the developer or visit:
